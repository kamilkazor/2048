# :two: 2048
Clone of 2048 game. You can check it out on https://kamilkazor.github.io/2048/
## Controls:
#### KEYBOARD:
ARROW KEYS: moving tiles, selecting button;  
ENTER: confirm;  
N: new game;
#### MOUSE:
You can move tiles by clicking and dragging.
#### TOUCHSCREEN:
Move tiles by touch and slide.
